Differentiation

TAG_Compare Description
hr & br hr จะมีการขีดเส้นแต่ br จะเป็นการเว้นวรรคอย่างเดียว
ul & ol ul จะเป็น list แบบใช้สัญลักษณ์ แต่ ol จะเป็น list แบบ Running Number
select & checkbox & radio button Select จะเป็น TAG ของตัวเองเลย ซึ่งจะเป็นการแสดงข้อมูลแบบ DropDown แต่ Checkbox กับ Radio จะเป็น Attribute ของ Input ซึ่ง Checkbox สามารถเลือกได้ทุกตัวแต่ Radio จะเลือกได้แค่ตัวเลือกเดียวภายในกลุ่มของปุ่มนั้นๆ
div & span div จะขึ้น line ใหม่ แต่ span จะยังอยู่ใน line เดิม
script & style script ใช้กับ JavaScript และ style ใช้กับ css
