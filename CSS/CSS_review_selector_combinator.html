CSS syntax ประกอบด้วย
Selector Declaration Property Value
การเขียน CSS มีอะไรบ้าง
External เป็นการเขียนแยกไฟล์ออกมาเลย
Internal เป็นการเขียนอยู่ด้านบนของ Page เดียวกัน
Inline เป็นการเขียนเฉพาะ TAG นั้นๆ
Selector มีอะไรบ้าง
ประเภท Selector ความหมาย ตัวอย่าง
Class selector เป็นการเรียกโดยมีกระบุชื่อ Class ให้กับ TAG .myclass{ font: bold 12px Tahoma;color: #009; }
ID selector เป็นการเรียกโดยมีกระบุชื่อ ID ให้กับ TAG นั้นๆ #myclass{ font: bold 12px Tahoma;color: #009; }
TAG selector เป็นการกำหนดค่าของ TAG ประเภทนั้นๆทั้งหมดเลย h1{ font: bold 12px Tahoma;color: #009; }
Combinator มีอะไรบ้าง
ประเภท Combinator ความหมาย ตัวอย่าง
Descendant ฉombinators เป็นการเรียกTAGที่เป็นประเภทเดียวกับ Selector ตัวที่2ทั้งหมดที่อยู่ภายใต้ Selector ตัวแรก .myclass div
Child Combinator เป็นการเรียก TAG เฉพาะลูกๆ ที่เป็นประเภทเดียวกับ Selector ตัวที่2 เท่านั้น .myclass > div
Adjacent Sibling Combinator เป็นการเรียก TAG ที่อยู่ถัดไป และ เป็นประเภทเดียวกับ Selector ตัวที่2 แค่ตัวเดียวเท่านั้น .myclass + div
General Sibling Combinator เป็นการเรียก TAG ที่อยู่ถัดไป และ เป็นประเภทเดียวกับ Selector ตัวที่2 ทั้งหมด .myclass ~ div
