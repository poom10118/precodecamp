Grid และ Flex ต่างกันอย่างไรบ้าง
grid จะเป็นการกำหนดรูปแบบ แบบตาราง
flex จะเป็นการกำหนดรูปแบบ แบบยืดหยุ่น
Justify และ Align ต่างกันอย่างไรบ้าง
Justify จะใช้จัดการกับแนวนอน
Align จะใช้จัดการกับแนวตั้ง
